# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar

```
yarn install
```

3. Tener Next CLI instalado

```
npm i -g @nestjs/cli
```

4. Levantar Base de datos

```
docker-compose up -d
```

5. Reconstruir la base de datos con la semilla
```
htpp://localhost:3000/api/v2/seed
```
